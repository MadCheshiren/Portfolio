# Project Documentation

**Developer:** Kirren Michael Fraginal  
**Project:** Professional Portfolio Website  
**Date:** April 2025

---

## Overview

A responsive single-page portfolio website built with HTML, Tailwind CSS, and JavaScript. Features a dramatic entrance animation with Tegaki handwriting, video background, dark/light theme toggle, and interactive project showcases.

---

## Features

### Entrance Animation
- Black screen overlay with Tegaki handwriting animation writing "Kirren"
- Video paused until user clicks
- Video starts from beginning on click
- Hero content reveals at 1.2 second splash moment
- Shows only on first visit (saved in localStorage)

### Navigation
- Floating pill-style navbar with smooth scroll
- Theme toggle (dark/light mode)
- Active section highlighting

### Projects
- **Task Manager:** Todo app with local storage
- **Dev Life:** Visual novel game about developer life
- **Form Validator:** Real-time form validation

### Interactive Elements
- 3D tilt effect on project cards
- Interactive terminal with commands
- Contact form via Formspree
- Scroll to top button

---

## Technologies

- **HTML5** - Structure
- **Tailwind CSS** - Styling
- **JavaScript (ES6+)** - Interactivity
- **GSAP** - 3D animations
- **Tegaki** - Handwriting animation
- **Formspree** - Form handling
